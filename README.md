# 우주에서온 콩콩이 (Space Kongkongi)

2D 싱글 스크린 아케이드 플랫포머. 적을 밟아 처치하고 웨이브를 생존하며 최고 점수를 달성하세요.

## 플레이

- **이동**: ← → / A D
- **점프**: ↑ / W / Space
- **모바일**: 하단 터치 버튼

## 게임 규칙

- 적을 **위에서 밟으면** 처치 + 바운스
- **콤보**: 착지 없이 연속 밟기 → 점수 배율 (×2, ×3, ×5, ×8)
- **아이템 블록**: 아래에서 머리로 치면 아이템 출현
- **웨이브**: 전멸 시 다음 웨이브, 난이도 점진적 상승
- 체력 0 → 게임오버

## 적 종류

| 적 | 특징 |
|---|---|
| 슬라임/벌레/쥐/무당벌레 | 지상 이동, 밟아서 처치 |
| 달팽이 | 2단계 처치 (껍질 → 파괴) |
| 벌/파리 | 공중 이동 |
| 톱날/가시슬라임 | 밟기 불가, 회피만 가능 (별 아이템으로 처치 가능) |
